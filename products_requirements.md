# 상품 목록 페이지 요구사항 분석 및 기능 목록

## 📌 기본 정보
### 프로젝트명: 
슈킹 패션 쇼핑몰 신발 상품 목록 페이지 (모바일)

### 사용하게 될 기술: 
React

### 문서 작성자: 김예진

## 📝 고객 요구사항 정리
- 젊은 세대가 주로 모바일을 사용하기 때문에 모바일에서의 환경 중시
- 신발 사진이 잘 보이게 작업
- 장바구니 상품 추가 시, 상품 개수 실시간 표시
- 작업 결과물을 확인할 수 있는 테스트 URL 제공 (2주 뒤)

## 📋 기능 목록
- 상품 리스트
- 상품을 2개의 컬럼으로 보여준다.
  - `<ProductList />`
- 상품 헤더 이미지 메인 화면에 표시
- 장바구니 갯수 표기
- 무한 스크롤 기능 구현
