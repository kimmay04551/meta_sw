# 상품 목록 페이지 프로젝트

## 프로젝트 개요

- 모바일에 최적화 된 상품 페이지, 장바구니 기능
- 김예진


## 개발 환경

React 17.0, JavaScript ES6

## 설치 및 실행 방법

```
npm install
npm start
```

## 컴포넌트 구조

- `ProductList`: 상품 목록을 보여줍니다.
- `ProductCard`: 각 상품의 정보를 보여줍니다.

## 테스트 방법

```
npm test
```

## 유의사항 및 알려진 이슈

무한 스크롤 기능이 없습니다. 스크롤 관련 오류 발생
